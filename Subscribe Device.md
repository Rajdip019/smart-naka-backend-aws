## Subscribe Device : ( Need to be updated from the camera system )

### `/subscribe-device`

Body

```json
{
  "topicName": The Police station name,
  "deviceToken" : //
}
```

Return :

```json
{
    "statusCode": 200,
    "body": {
      "deviceEndPoint": "endpointArn",
      "topicName": "event.topicName",
    }
  }
```
